##### 비트 연산자와 기타 연산자

###### 비트 연산자

- "<<"는 비트를 왼쪽으로 이동(shift)시킨다.
- ">>"는 비트를 오른쪽으로 이동(shift)시킨다.
- 예시: b = a << 2 -> a의 값을 왼쪽으로 2비트 이동시킨 결과를 b에 저장한다. (2진수)

##### 기타 연산자

- size of 연산자
- c언어에서 변수, 변수형, 배열의 저장 장소의 크기를 byte단위로 구한다.
- 예시 : printf("int 자료형의 크기 : %d, sizeof(int)); 결과 int 자료형의 크기 : 4  

- 콤마 연산자
- 성격이 동일한 자료형을 나열할 때 사용된다.
- 예시 : int a,b,c;

- cast 연산자(형 변환 연산자)
- 명시적 형 변환 시에 사용하며 어떤 수식을 다른 데이터형으로 변경할 때 사용한다.
- 예시 : int num = (int)3.14 + 5; 결과 : 실수형 상수 3.14를 정수형상수 3으로 변환후 3+ 5를 수행 8이란 결과값을 num에 대입.

##### 포인터 연산자

- c언어의 단항 연산자 중, &는 변수의 주소를 의미하고 *는 변수의 내용을 의미한다.
- 예시 int a = 3; int *ptr + 5; 결과 정수형 변수 a를 포인터 변수 ptr이 참조하고 있고 *ptr은 포인터 변수에 포인터 연산자를 통해 값을 참조하여 변수 a의 내용인 3을 5와 덧셈하여 8의 결과를 포인터 변수 ptr을 통해 다시 a의 내용으로 대입한다.
